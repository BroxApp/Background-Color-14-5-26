# Random Background Color Changer

A simple and lightweight web application that changes the page's background color to a random hexadecimal value every time a button is clicked.

## Features
- Generates a random hexadecimal color code.
- Dynamically updates the `body` background style.
- Clean and minimal JavaScript implementation.

## How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Click the button to see the background color change!

## Technologies Used
- HTML / CSS / Vanilla JavaScript
